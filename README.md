# New Features !

  - Add ARMv8 arch

#### Install
You need Linux distribution like Ubuntu or Manjaoro

```sh
git clone https://github.com/Bensuperpc/docker-tinycc.git
```
#### Build
```sh
cd docker-tinycc && make linux/amd64
```
#### Usage

```sh
./tinycc.sh tcc hello.c
```
